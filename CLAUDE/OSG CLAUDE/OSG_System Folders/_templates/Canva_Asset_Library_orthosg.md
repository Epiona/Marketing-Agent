## Zweck dieses Dokuments

Dieses Dokument listet alle Canva-Designs der Orthopädie St. Gallen vollständig auf, gegliedert nach Ordner. Es dient als Referenz für den Claude Marketing Agenten, damit er:

- Bestehende Templates erkennt und wiederverwenden kann

- Bei neuen Aufgaben auf passende Vorlagen verweisen kann
- Via Canva MCP direkten Zugriff auf jedes Design hat (Canva Design-ID angegeben)

**Canva-Ordner: Orthopädie St. Gallen | Canva-Verbindung: MCP (mcp.canva.com)**

Für Claude Code gilt: Den Canva MCP Server einbinden (URL: https://mcp.canva.com/mcp), dann ist der direkte Zugriff auf alle Designs über die angegebenen Design-IDs möglich — ohne lokale Dateien.

## Hinweis für Claude Code

Du kannst Claude Code direkten Zugang zu allen Canva-Designs geben — ohne lokale Template-Ordner auf deinem Computer.

**So richtest du den Canva MCP in Claude Code ein:**

- Claude Code installieren (Terminal: `npm install -g @anthropic-ai/claude-code`)
- Canva MCP Server hinzufügen: In der Claude Code Konfiguration den Server `https://mcp.canva.com/mcp` eintragen
- Einmalig mit Canva-Account authentifizieren
- Danach kann Claude Code alle Designs über die Canva Design-IDs in diesem Dokument direkt ansprechen

**Was Claude Code dann kann:**

- Designs lesen: `get-design-content` mit der Design-ID
- Texte ersetzen: `start-editing-transaction` → `perform-editing-operations` → `commit`
- Designs exportieren: `export-design` als PDF, PNG oder MP4
- Neue Designs aus Templates generieren

**Workflow für Post-Generierung:**

1. Agent wählt passendes Template aus dieser Library
2. Agent generiert Caption + Slide-Texte gemäss Brand Voice Guide
3. Agent ruft Canva MCP auf → ersetzt Platzhalter-Texte mit neuem Content
4. Export als PNG für Instagram
5. Caption + Hashtags separat ausgeben

Vorteil gegenüber lokalem Ordner: Änderungen in Canva sind sofort sichtbar. Kein manuelles Hoch- und Herunterladen. Claude Code arbeitet direkt in deinen Live-Designs.
