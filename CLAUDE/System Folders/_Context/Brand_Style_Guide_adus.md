**2.3.3 Color Usage Rules (STRICT)**

**1. Hierarchy of Colors**

1.  Neutral backgrounds (grey/white)

2.  Primary system color (depending on context)

3.  Secondary soft tones

4.  Accent (lime) ONLY for highlights

**2. Do NOT:**

-   Mix all color systems randomly

-   Overuse the lime accent

-   Use bright or external colors

-   Create high contrast chaos

**3. Context-Based Color Logic**

  -----------------------------------------------------------------------
  **Context**                                     **Color**
  ----------------------------------------------- -----------------------
  Doctors / Fachpersonen                          Red

  Navigation / Locations                          Green

  Specialists                                     Blue

  Highlights                                      Lime

  Background                                      Grey/White
  -----------------------------------------------------------------------
