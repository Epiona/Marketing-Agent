# Zweck dieses Dokuments

Dieses Dokument listet alle Canva-Designs der ADUS Klinik vollständig
auf, gegliedert nach Ordner. Es dient als Referenz für den Claude
Marketing Agenten, damit er:

-   bestehende Templates erkennt und wiederverwenden kann

-   die visuelle Sprache und Content-Formate versteht

-   bei neuen Aufgaben auf passende Vorlagen verweisen kann

-   via Canva MCP direkten Zugriff auf jedes Design hat (Canva Design-ID
    angegeben)

**Canva-Ordner: ADUS Klinik \| Canva-Verbindung: MCP (mcp.canva.com)**

Für Claude Code gilt: Den Canva MCP Server einbinden (URL:
https://mcp.canva.com/mcp), dann ist der direkte Zugriff auf alle
Designs über die angegebenen Design-IDs möglich --- ohne lokale Dateien.

# Hinweis für Claude Code

Du kannst Claude Code direkten Zugang zu allen Canva-Designs geben ---
ohne lokale Template-Ordner auf deinem Computer.

**So richtest du den Canva MCP in Claude Code ein:**

-   Claude Code installieren (Terminal: npm install -g
    \@anthropic-ai/claude-code)

-   Canva MCP Server hinzufügen: In der Claude Code Konfiguration den
    Server https://mcp.canva.com/mcp eintragen

-   Einmalig mit Canva-Account authentifizieren

-   Danach kann Claude Code alle Designs über die Canva Design-IDs in
    diesem Dokument direkt ansprechen

**Was Claude Code dann kann:**

-   Designs lesen: get-design-content mit der Design-ID

-   Texte ersetzen: start-editing-transaction →
    perform-editing-operations → commit

-   Designs exportieren: export-design als PDF, PNG oder MP4

-   Neue Designs aus Templates generieren

Vorteil gegenüber lokalem Ordner: Änderungen in Canva sind sofort
sichtbar. Kein manuelles Hoch- und Herunterladen. Claude Code arbeitet
direkt in deinen Live-Designs.
